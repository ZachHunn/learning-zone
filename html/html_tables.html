HTML Tables

An HTML table is a structured way to display tabular data on a web page. It consists of rows and columns, forming a grid-like structure.

  • HTML Tables are used to display data in a tabular format, with rows and columns.
  • A table is created using the table element
    • <table></table>
  • The table head element defines the table header rows.
    • <thead></thead>
  • You can create table headings by using the th element.
    • <th></th>
  • The table body is used to group content of the table. It is usually sued to enclose a set of table rows that represent the main data of the table.
    • <tbody></tbody>
    • Although this tag is not strictly required it is considered good practice to use it for better structure and semantic markup.
  • Within the table element you can create rows using the tr element.
    • <tr></tr>
  • To create columns use the td element.
    • <table></table>

Spanning Rows and Columns

In HTML tables, the colspan and rowspan attributes are used to control the span or merging of cells across multiple columns or rows. These attributes allow you to create more complex table structures and merge cells to accommodate larger or grouped data.

  • colspan attribute: The colspan attribute specifies the number of columns a cell should span horizontally. It is applied to the <td> or <th> elements within the <tr> (table row) structure.
  • rowspan attribute: The rowspan attribute specifies the number of rows a cell should span vertically. It is also applied to the <td> or <th> elements within the <tr> structure.
