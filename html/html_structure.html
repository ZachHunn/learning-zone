HTML Document Structure

HTML (Hypertext Markup Language) structures a webpage by providing a hierarchical structure of elements that define the content and layout of the page.

Document Type Declaration (DOCTYPE) - <!DOCTYPE>

Doctype declaration tells the broswer that this file is an HTML5 document

HTML Element - <html>

The HTML element serves as the container of the rest of the HTML Elements. It has two main sections: head and body elements

Head Element - <head>

the head element contains information about the website that is not displayed on the webpage such as page titles, external files, and metadata

META Element - <meta>

Meta elements are used by web broswers, search engines, and other web services to better understand and process the content of an HTML document. They can impact on how they document is displayed, indexed, and ranked by search engines

Title Element - <title>

The title element specifies the title the HTML document

Body Element - <body>

The body element contains the content of the webpage that is displayed to the users such as text and images

Div Element - <div>

The div element is used to group elements together for styling and functionality

The Heading Elements - <h1> - <h6>

The heading elements are used tp define headings and subheadings on a webpage. The purpose is to provide a hierarchiral structure to the content to make it easier for humans and machinces to read

Paragraph Element - <p>

The Paragraph element is used to create paragraphs of text. The purpose is to group a block of text that represents a single concept.

Span Element - <span>

The span element is like a div for inline text. This is uesed to group inline text or other inline elements for styling purposes
