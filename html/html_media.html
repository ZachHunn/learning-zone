HTML Media Elements

HTML offers a range of media elements, including audio, image, and video, allowing for versatile multimedia integration.

Image Element - <img/>

  • The img tag is used to embed images into a web page.
  • You'll notice that the you don't have to wrap anything in the img tag. That's becaause it is a self-closing tag, which means it does not require a closing tag.
  • Most self closing tags use attributes to provide additional information to modify the behavior of the element.

Attributes that img tags use

  • src - This attribute specifies the URL or location of the image to be displayed. It can be a local file path or a URL.
  • alt - This specifies the alternate text that will be displayed in case the image fails to load or the user is unable to see the image. This is important for accessibility purposes.
  • width - This specifies the width of the image in pixels or as a percentage of the parent element's width.
  • height - This specifices the height of the image in pixels or as a percentage of the parent element's width.

Video Tag - <video></video>

  • The video tag is used to embed video content into an HTML document.
  • The content in between the video element is a fallback message that displays if the broswer does not support the video tag.

Attributes used for video element

  • src - specifices the local file path or URL
  • contols - adds playback controls
  • loop - loops the video
  • muted - mutes the video
  • autoplay - auto plays the video

Auido Element - <audio></audio>

  • Audio element is used to embed audio into an HTML Document
  • Audio element share many of the same attributes as the video tag

Source Element - <source>

  • The source element is used within the video or audio tags to specify alternative video or auiso sources.
  • It is used to provide different formats and quaility levels of the media file for browser compatibility.
  • Multiple source elements can be used with the video or audio element
