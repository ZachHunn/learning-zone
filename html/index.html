HTML - Hypertext Markup Language

What is HTML?

  • HTML or Hypertext Markup Lanaguage is the standard markup language for creating and structuring web pages and applications.

HTML Document

  • The HTML document serves as the foundation for creating web pages and applications. It is a text file written in HTML markup language that contains the structure, content, and presentation instructions for a web document.
  • The HTML document is responsible for defining the elements, layout, and formatting of the content to be displayed in a web browser.
  • Once the HTML document is loaded by the browser, it is transformed into a data structure called the Document Object Model (DOM).
  • By leveraging the DOM, developers can dynamically update the web page, respond to user interactions, change styles, add or remove elements, and perform other actions to create interactive and responsive web applications
  • The DOM serves as a powerful interface that bridges the static HTML document and the dynamic capabilities provided by scripting languages like JavaScript. Which will be covered later

HTML Elements and Tags

HTML Element

An HTML element represents a specific type of content within an HTML document. It defines the structure and semantics of that content. An HTML element consists of an opening tag, optional content, and a closing tag (in most cases). The opening tag marks the beginning of the element, the closing tag marks the end, and the content is the actual content or nested elements contained within the element.

HTML Tag

An HTML tag is the markup syntax used to define and enclose HTML elements within an HTML document.

  • HTML Tags are enclosed in angle brackets (< >).
  • HTML uses sets of tags and attributes to define the structure and presentation of content on a web page.
  • Most HTML tags have an opening and closing tag.
    • Opening tag (<div>) - This identifies the type of HTML element that is being defined.
    • Closing tag (</div>) - This makes the end of an element. Together opening and closing tags define the boundries of an HTML element and any content that falls within it.
  • Some HTML tags, called self-closing tags or void elements, do not have a closing tag and are written in a specific self-closing format, such as the <img/> tag.
  • HTML tags are used to mark up different elements such as headings, paragraphs, links, imagesm tables, forms, and more.
  • An HTML attribute provides additional information or modifies the behavior of an HTML element. Attributes are specified within the opening tag of an HTML element and consist of a name and a value, separated by an equals sign.
