HTML Inputs

HTML inputs are elements that enable user interaction and data input on a web page. They provide a way for users to enter or select information and submit it to a server for processing. Inputs are used in various forms, surveys, search bars, and other interactive components.

Label Element - <label></label>

The label element ia used to associate a label an HTML form element typically an input element.

Text Input - <input type="text"/>

Text input allows users to enter free-form text.

Password input - <input type="password"/>

Password input conceals the entered text to provide secure password entry.

Checkbox Input - <input type="checkbox"/>

Checkbox input presents a checkbox that users can select or deselect.

Radio Button Input - <input type="checkbox"/>

Radio button input displays a set of options where users can select only one choice.

Select Input - <select></select>

Select input creates a dropdown menu with a list of options, and users can select one or multiple choices.

Option Element - <option></option>

The option element in HTML is used within a <select> element to define individual options within a dropdown list. It represents an option that users can select from.

Input Button - <input type="button" value="Click me/>

Input buttons in HTML are interactive elements that allow users to trigger actions or submit forms within a web page. The <input> element with the type="button" or type="submit" attribute is used to create a button.
