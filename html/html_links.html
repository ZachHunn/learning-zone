HTML Links

Anchor Tag - <a>

  • Anchor tags also known as hyperlinks are elements used to create clickable links.
  • They allow users to navigate between different web pages, sections within the same page, or an external resource
  • The <a> tag is used to define an HTML link
  • It requires a 'href' attribute that specifies the destination URL or target location of the link
  • When a user clicks on an HTML link, the broswer rediercts them to the specified URL

Use cases

Anchor tags can be used to link to various destinations, including:

  1. Web Pages: Specify the URL of another web page to link to different websites or pages within the same website.
  2. Anchors: Use anchor tags (<>) with href values that correspond to element IDs within the same page to create in-page navigation.
  3. Email Addresses: Use the mailto: scheme followed by an email address to open the user's email client with a new message.
  4. File Downloads: Specify the path to a file to enable users to download files, such as documents, images, or multimedia files.

Link Tag - <link>

  • Although it shares the name "link," the <link> is not directly related to creating hyperlinks like the <a> tag.
  • It is used to include external resources and define relationships between HTML documents
  • It is commonly used within the <head&> section of an HTML document to reference external CSS stylesheets, web fonts, or other external resources
  • It uses the 'href' attribute to specifiy the path or URL or the external resource and may include additional attributes depending on the resource being linked
