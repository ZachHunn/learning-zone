Using Colors In CSS

CSS color properties determine the appearance of color-related aspects within an element.

How to specify colors

  • Keyword Colors: CSS provides a set of predefined color names like "red," "blue," or "green."
  • Hexadecimal Colors: Colors can be specified using a six-digit hexadecimal code, such as #RRGGBB. Each pair of digits represents the intensity of red (RR), green (GG), and blue (BB) respectively.
    • #FF0000 represents pure red.
  • RGBA Colors: RGBA is similar to RGB, but with an additional alpha channel that defines the transparency of the color.
    • The alpha value ranges from 0 (fully transparent) to 1 (fully opaque)
    • rgba(255, 0, 0, 0.5) represents semi-transparent red.
  • HSL and HSLA Colors: HSL stands for Hue, Saturation, and Lightness. The hue represents the color, saturation controls its intensity, and lightness determines how light or dark it appears.
    • HSL colors are defined using the hsl() function, while HSLA includes an alpha channel for transparency.
  • System Colors: CSS also supports a set of system colors like "window," "buttonface," or "highlight," which can be used to match the colors of the user's operating system or browser theme.

Color Properties

CSS color properties determine the appearance of color-related aspects within an element.

  • color: Sets the color of the text content within an element. It affects the color of the text characters.
  • background-color: Sets the background color of an element. It affects the color of the area behind the content, such as the background of a div or the entire page.
  • border-color: Sets the color of the borders of an element. It affects the color of the lines that surround the element.
  • outline-color: Sets the color of the outline around an element. It affects the color of the outline, which is a visual highlight that appears outside the borders of an element.
  • box-shadow: Sets the color of a box shadow. It affects the color of the shadow that appears behind an element, creating a visual effect of depth or elevation.
  • text-shadow: Sets the color of a text shadow. It affects the color of the shadow that appears behind the text characters, adding a visual effect of depth or emphasis.
  • fill: Sets the color used to fill the content of SVG (Scalable Vector Graphics) elements. It affects the color of shapes or paths within an SVG image.
  • stroke: Sets the color used to draw the outlines of SVG elements. It affects the color of the lines that form the edges of shapes or paths within an SVG image.

Changing Foreground Color Ex:

            h1 {
              color: rgba(255, 0, 0, 0.5); /* RGBA */
              color: hsla(0, 100%, 50%, 0.5); /* HSLA */
              color: red; /* Color Name */
              color: #FF0000; /* Hexadecimal */
            }

Chaning Background Color Ex:

            body {
              background-color: rgba(0, 128, 0, 0.75); /* RGBA */
              background-color: hsla(120, 100%, 25%, 0.75); /* HSLA */
              background-color: green; /* Color Name */
              background-color: #008000; /* Hexadecimal */
            }
