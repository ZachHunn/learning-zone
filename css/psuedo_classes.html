What Are Psuedo Classes and Elements?

Psudeo Classes

Pseudo-classes are used to select elements based on a specific state or condition. They are preceded by a colon (:) and follow the selector

Pseudo-classes are used to modify the styling of elements based on user interactions, document structure, or dynamic states.

  • `:hover`: Matches an element when the user hovers over it with the cursor.
  • `:active`: Matches an element when it is being activated or clicked.
  • `:focus`: Matches an element when it has received focus (e.g., through keyboard navigation or clicking).
  • `:visited`: Matches a visited link.
  • `:first-child`: Matches the first child element of its parent.
  • `:last-child`: Matches the last child element of its parent.
  • `:nth-child()`: Matches elements that are specified by a formula.

Psudeo Elements

Pseudo-elements allow you to style specific parts of an element, such as the first letter or line of text within a paragraph. They are also preceded by a double colon (::) or, in some cases, a single colon (:).

Pseudo-elements allow you to add decorative elements, modify specific parts of the content, or apply special styles to selected portions of an element.

  • `::before`: Inserts content before the selected element.
  • `::after`: Inserts content after the selected element.
  • `::first-letter`: Selects the first letter of the text content within an element.
  • `::first-line`: Selects the first line of the text content within an element.
  • `::selection`: Selects the portion of text that has been highlighted by the user.

Pseudo-classes and pseudo-elements provide powerful ways to apply styles to specific states, conditions, or portions of elements. They enhance the flexibility and expressiveness of CSS, allowing you to create dynamic and interactive styles for your web pages.
