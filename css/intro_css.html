CSS - Cascading Style Sheets

What is CSS?

  • CSS, which stands for Cascading Style Sheets, is a language used to describe the presentation and visual style of a web page.
  • It works alongside HTML to define how elements should be displayed on the screen, including their colors, fonts, sizes, positions, and other visual properties.
  • CSS allows web developers to create consistent and visually appealing designs, customize the look of different elements, and ensure a better user experience. It provides flexibility and control over the appearance of web content, helping to create beautiful and engaging websites.

EXternal Vs Internal CSS

CSS can be used with HTML both internally, by embedding the CSS code within the HTML file, and externally, by linking an external CSS file to the HTML document.

Internal CSS

  • Styles are written within the <style> tags in the <head> section of the HTML file
  • It keeps the styles within the HTML file itself, suitable for small-scale or single-page projects.

      • Inline CSS

    • Inline CSS involves directly applying styles to individual HTML elements using the style attribute.
    • It allows for individual styling of specific elements, but can lead to code duplication and reduced maintainability when applied to multiple elements.

External CSS

  • External CSS involves creating a separate CSS file with a .css extension that contains all the CSS code.
  • The CSS file is then linked to the HTML document using the <link> tag within the <head> section.
  • External CSS allows you to keep your CSS code separate from your HTML code. This separation promotes better organization, reusability, and easier maintenance of styles across multiple web pages.

How CSS Works

  • CSS works with HTML by applying CSS rules to HTML elements. These rules govern how the content of specified elements should be displayed in a browser.
  • CSS rules cascade or flow from the top of the document to the bottom. When multiple CSS rules target the same element or property, the rule with the highest specificity and the most recent declaration takes precedence.
  • This allows you to define styles at different levels, such as in an external stylesheet, internal stylesheet, or inline style, and have them apply to elements accordingly.
  • CSS rules consist of two parts: a selector (red) and a declaration (green). 
            h1  {
              font-size: 28px
            }
  • In CSS, inheritance is a mechanism that automatically passes certain properties from parent elements to their child elements.
  • This means that styles applied to a parent element can be inherited by its children, unless explicitly overridden.
  • It allows for efficient and consistent styling, as you can define styles at higher levels and have them propagate down the document tree to affect multiple elements.

CSS Selectors

  • Selectors are used to target specific HTML elements that you want to style.
  • They can be based on element names, classes, IDs, attributes, or other criteria.

Types of Selectors

  • Element selectors: Select elements based on their tag names.
    • 'h1' selects all <h1> tags.
  • Class selectors: Select elements based on their class attribute. The selector starts with a dot (.) followed by the class name.
    • .my-class selects all elements with the class name "my-class".
  • ID selectors: Select a single element based on its ID attribute. The selector starts with a hash (#) followed by the ID name.
    • #my-id selects the element with the ID "my-id".
  • Attribute selectors: Select elements based on their attribute values. They can be used to select elements with specific attribute values or attribute presence.
    • [type="submit"] selects elements with the attribute "type" set to "submit".
  • Descendant selectors: Select elements based on their hierarchical relationship to other elements.
    • div p selects all <p> elements that are descendants of a <div> element.
  • Child selectors: Select elements that are direct children of another element. The selector uses the greater-than symbol (>).
    • ul > li selects all <li> elements that are direct children of a <ul> element.
  • Adjacent sibling selectors: Select the next sibling element that directly follows another element. The selector uses the plus sign (+).
    • h2 + p selects the <p> element that directly follows an <h2> element.
  • General sibling selectors: Select sibling elements that follow another element. The selector uses the tilde (~).
    • h2 ~ p selects all <p> elements that follow an <h2>element.
  • Pseudo-classes: Select elements based on their state or position in relation to user interactions or document structure.
    • :hover selects an element when the user hovers over it.
  • Pseudo-elements: Select and style a specific part of an element. They are denoted by double colons (::)
    • ::before inserts content before an element.
