Typography

Typography in CSS refers to the art and technique of arranging and styling text to make it visually appealing and readable on web pages. CSS provides several properties that allow you to control various aspects of typography.

In CSS (Cascading Style Sheets), you can style text by applying various properties and values to the elements containing the text. Here are some common ways to style text in CSS:

Font Properties

  • font-family: Sets the font face for the text.
  • font-size: Specifies the size of the font.
  • font-weight: Defines the thickness or boldness of the font.
  • font-style: Specifies the style of the font, such as italic or normal.

Text Color and Background

  • color: Sets the color of the text.
  • background-color: Defines the background color behind the text.

Text Alignment and Spacing

  • text-align: Aligns the text within its container, such as left, center, or right.
  • line-height: Sets the height of each line of text.
  • letter-spacing: Adjusts the spacing between characters.
  • word-spacing: Adjusts the spacing between words.

Text Decoration

  • text-decoration: Adds visual effects to the text, like underline, overline, or strike-through.
  • text-decoration-color: Sets the color of the text decoration.
  • text-decoration-line: Specifies the type of decoration, such as underline, overline, or line-through.

Text Transformation

  • text-transform: Transforms the text, such as making it uppercase, lowercase, or capitalize.

Text Shadow

  • text-shadow: Adds a shadow effect behind the text. It takes values for horizontal offset, vertical offset, blur radius, and color.

Text Overflow

  • overflow: Determines how the text should be handled when it exceeds the size of its container.
  • white-space: Specifies how white spaces should be treated, such as preserving or collapsing them.
